# Expedition Checklist

## Before
- [ ] Define a clear topic or question
- [ ] Invite participants (5–7 people)
- [ ] Create a repository using this expedition-template

## During
- [ ] Focus on hands-on work
- [ ] Capture key observations in the logbook
- [ ] Save useful artifacts (code, prompts, screenshots)

## After
- [ ] Review and complete the logbook (README)
- [ ] Add links to artifacts / repos / PRs
- [ ] Write 1–2 sentence summary
